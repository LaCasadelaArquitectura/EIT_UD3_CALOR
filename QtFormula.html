Ejemplo cálculo de calor Latente




Para calcular el calor total podemos hacerlo mediante 2 métodos:

  • 1º Mediante la suma de calor sensible QS y calor latente QL :

    - QS = Ce x Pe x V x Δ temperatura
    - QL = Cv x Pe x V x Δ humedad
    - QT = QS + QL

    - QS = 0.24 x 1.24 x 150 x (40º - 19º) = 937.44 Kcal
    - QL = 0.6 x 1.24 x 150 x 10g = 1116 Kcal
    - QT = 937.44 + 1116 = 2053.44 Kcal

  • 2º Mediante la formula de la entalpía :

    - QT = Pe x V x Δ Entalpía

    - Δ Entalpía= 92.5 Kj - 45 Kj = 47.5 Kj
    - 47,5Kj = 11.35 Kcal

    - QT = 1.24 x 150 x 11.35 = 2111 Kcal

La pequeña diferencia entre ambos resultados se debe unicamente a pequeños errores al dibujar
